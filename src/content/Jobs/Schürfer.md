Der Schürfer ist der Bergbau-Job: Baue Stein und Erze ab für Job-XP und Pfund.

## Wie verdienst du XP?

| Block | Job-XP |
|---|---|
| Stein | 0,78 |
| Tiefenschiefer | 1,4 |
| Kohle | 2,5 |
| Nether-Quarz | 2,0 |
| Lapis | 4,2 |
| Redstone | 4,4 |
| Eisen | 4,0 |
| Gold | 4,8 |
| Obsidian | 7,0 |
| Smaragd | 9,0 |
| Diamant | 12,0 |
| Alter Schutt | 24,0 |

## Fähigkeiten

Über Verzauberungen freischaltbar:

- **Drill**, **Vein-Mining** (Ader-Abbau), **Schmelz-Berührung** (Erze direkt zu Barren), **Magnet** (Drops einsammeln).
