Der Citybuild ist dein eigenes Stück Land auf dem Server. Beantrage ein Plot, bau dein Heim und gestalte es ganz nach deinen Vorstellungen.

## Plots

Im Citybuild bekommt jeder Spieler ein eigenes **Plot** — ein abgegrenztes Grundstück, auf dem nur du (und eingeladene Spieler) bauen und verändern kannst.

Wichtige Plot-Befehle:

| Befehl | Funktion |
|---|---|
| `/plot auto` | Automatisch ein freies Plot zuweisen lassen |
| `/plot claim` | Das Plot, auf dem du stehst, beanspruchen |
| `/plot home` | Zu deinem Plot teleportieren |
| `/plot add <Spieler>` | Spieler zum Bauen einladen (bei dir offline möglich) |
| `/plot trust <Spieler>` | Spieler als vollständig vertrauenswürdig hinzufügen |
| `/plot remove <Spieler>` | Spieler vom Plot entfernen |
| `/plot info` | Infos über das aktuelle Plot anzeigen |
| `/plot merge` | Zwei angrenzende Plots zusammenführen |
| `/plot clear` | Plot vollständig zurücksetzen |
| `/plot delete` | Plot löschen und freigeben |

## Plot-Flags

Plot-Flags sind Einstellungen, die das Verhalten deines Plots steuern. Sie werden über ein **GUI** verwaltet. Verschiedene Flags sind je nach deinem **Server-Rang** freigeschaltet:

- Items Dropbar
- Redstone
- Eisschmelze / Schneeschmelze
- Rankenwachstum
- ChestShop Hopper
- Plot-Titel

## Kistenshops im Citybuild

Du kannst auf deinem Plot **Kistenshops** einrichten und Waren an andere Spieler verkaufen oder ankaufen. Mehr dazu in der [Kistenshops-Anleitung](#/wiki/anderes/kistenshops).
