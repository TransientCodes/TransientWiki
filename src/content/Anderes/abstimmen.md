Durch das Abstimmen für TransientRealm auf Server-Listen unterstützt du den Server und erhältst dafür eine kostenlose **Vote-Kiste**.

## So funktioniert es

1. Gib im Spiel den Befehl `/vote` ein.
2. Du erhältst einen Link zu den Vote-Seiten im Chat.
3. Stimme auf den verlinkten Seiten für den Server ab.
4. Du erhältst automatisch eine **Vote-Kiste** in dein Inventar.

## Belohnung

Jeder Vote belohnt dich mit einer **Vote-Kiste**, die verschiedene nützliche Items und Pfund enthält. Kisten können am Kisten-Bereich am Spawn geöffnet werden.
