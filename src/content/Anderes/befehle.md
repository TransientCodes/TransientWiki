## Allgemein

| Befehl | Beschreibung |
|---|---|
| `/spawn` | Zum Spawn teleportieren |
| `/warp <Name>` | Zu einem Spieler-Warp teleportieren |
| `/farmwelt` | In die Farmwelt teleportieren |

## Plots (Citybuild)

| Befehl | Beschreibung |
|---|---|
| `/plot auto` | Automatisch ein freies Plot zuweisen lassen |
| `/plot claim` | Das Plot, auf dem du stehst, beanspruchen |
| `/plot home` | Zu deinem Plot teleportieren |
| `/plot add <Spieler>` | Spieler zum Bauen einladen |
| `/plot trust <Spieler>` | Spieler vollen Zugriff auf das Plot geben |
| `/plot remove <Spieler>` | Spieler vom Plot entfernen |
| `/plot info` | Infos zum aktuellen Plot anzeigen |
| `/plot merge` | Zwei benachbarte Plots zusammenführen |
| `/plot clear` | Plot vollständig zurücksetzen |
| `/plot delete` | Plot löschen und freigeben |

## Kistenshops

| Befehl | Beschreibung |
|---|---|
| `/trustshop <Spieler>` | Spieler Zugriff auf den Shop-Container geben / entziehen |
| `/shareincome <Spieler> <Prozent>` | Einnahmen prozentual mit einem Spieler teilen |
| `/shopinfo` | Informationen über einen Shop anzeigen |
